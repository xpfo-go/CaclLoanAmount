# 房贷计算器 Web 版需求（React）

## 1. 项目目标
- 将仓库从 Go CLI 迁移为前端 Web 项目。
- 所有计算在浏览器本地完成，无后端依赖。
- 提供可视化、可交互、对新手友好的学习型代码结构。
- 通过 GitHub Pages 发布在线体验页面。

## 2. 用户场景
- 普通用户希望快速比较公积金贷款、商业贷款、组合贷的月供和总利息。
- 用户希望支持浮动利率重定价（按月份分段）。
- 用户希望通过图表直观看到月供变化趋势。

## 3. 功能需求
- 输入项：
  - 房屋总价（万元）
  - 首付/本金（万元）
  - 公积金贷款金额（万元）
  - 公积金贷款年限、年利率（当公积金额 > 0 时）
  - 商贷年限、商贷初始年利率（当商贷金额 > 0 时）
  - 还款方式：等额本息 / 等额本金
  - 商贷重定价（格式：`13:3.2,25:3.1`）
- 输出项：
  - 公积金月供、商贷月供、组合月供
  - 公积金总利息、商贷总利息、组合总利息
- 图表：
  - 月度本金/利息堆叠柱状图
  - 剩余本金趋势图
  - 仅公积金或仅商贷时显示 1 套图表
  - 组合贷时可切换合并/公积金/商贷视图
- 交互能力：
  - 示例一键填充
  - 输入校验与错误提示
  - 结果卡片与图表联动更新

## 4. 非功能需求
- 代码清晰、模块化，适合 React 新手阅读。
- 具备基础单元测试（核心计算逻辑）。
- GitHub Actions 自动构建与部署到 GitHub Pages。
## 5. 迁移需求
- 仓库以 React Web 版本为唯一维护入口。
- 根目录主项目切换为 React 前端。
