# CaclLoanAmount

房贷计算器（Go），支持：
- 等额本息（`epi`）
- 等额本金（`ep`）
- 组合贷（公积金 + 商贷）
- 商贷按重定价月分段利率计算（例如 `13:3.2,25:3.1`）

## 计算口径

- 等额本息（月供）

`M = P * i * (1+i)^N / ((1+i)^N - 1)`

- 等额本金（第 k 期）

`A_k = P/N + (P - (k-1)*P/N) * i`

其中：
- `P` 为贷款本金
- `i` 为月利率（年利率/12）
- `N` 为总月数

对于分段利率（重定价），本工具在每个分段起点按“剩余本金 + 剩余期数 + 新利率”重新计算后续月供（等额本息模式）。

## 使用方式

```bash
go run .
```

按提示输入：
- 房屋总价（万元）
- 本金（万元）
- 公积金贷款额（万元）
- 公积金贷款额 > 0 时，再输入公积金年限和年利率
- 商贷金额 > 0 时，再输入商贷年限和初始年利率
- 还款方式（`epi` 或 `ep`）
- 商贷金额 > 0 时，可输入商贷重定价串（可空）

计算结果最后会附带 ASCII 折线图：
- 横轴：时间（月）
- 纵轴：金额（万元）
- 只有公积金或只有商业贷款时，输出 1 张图
- 公积金和商业贷款都有时，分别输出 2 张图

示例重定价串：
- `13:3.2`：第13个月起年利率 3.2%
- `13:3.2,25:3.1`：第13个月起 3.2%，第25个月起 3.1%
